Fitur Lengkap

Semua yang kamu butuhkan

Didesain untuk developer yang ingin rekap kerja tanpa ribet — dari fetch hingga Excel siap kirim.

Live Streaming

Commit muncul satu per satu secara real-time saat di-fetch. Progress bar menunjukkan berapa repo yang sudah selesai.

AI Enhancement

Bersihkan pesan commit berantakan jadi judul dan deskripsi tugas yang profesional menggunakan Gemini AI — satu klik saja.

Export Excel

Download laporan .xlsx berformat rapi — nomor, nama tugas, deskripsi, modul, tanggal, dan link commit.

Filter Repository

Pilih satu atau beberapa repository spesifik dari dropdown dengan searchbox — commit yang di-fetch hanya dari repo yang dipilih.

Filter Username

Masukkan username siapa saja untuk melihat commit mereka — tanpa perlu login atau tambah akun. Cocok untuk recap tim.

Multi-Akun

Tambahkan akun Gitea lain untuk menggabungkan repositorinya ke sidebar. Rekap dari beberapa akun dalam satu tampilan.

Edit & Hapus Baris

Edit nama tugas, deskripsi, dan modul langsung di tabel. Hapus baris yang tidak relevan sebelum export — kontrol penuh ada di kamu.

Filter Merge Commit

Satu centang untuk otomatis sembunyikan semua commit merge pull request dan merge branch dari daftar — tanpa hapus manual.

Riwayat Export

Semua file Excel yang pernah digenerate tersimpan otomatis. Download ulang atau hapus kapan saja dari halaman Riwayat Export.
